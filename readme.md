# Team Portfolio Website

## Description

An interactive team introduction website for SE2144 featuring a landing page and individual portfolio pages for each member. Designed to mimic the feel of fighting games, where each portfolio feels like a character you choose.

## Installation

1. Clone or download this repository.
2. Open `index.html` in your browser, or serve the folder with a local development server.
3. Make sure the `src/` assets folder stays in the same relative location so the images, fonts, and audio load correctly.

## Usage

1. Open the main page in `index.html`.
2. Click **PRESS START** to enter the site.
3. Choose a team member to view their introduction page and personal content.
4. Explore the individual portfolio pages in `joseph.html`, `sherwin.html`, and `xanth.html`.
